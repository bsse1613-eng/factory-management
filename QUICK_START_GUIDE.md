## â“ FAQ

**Q: Can I change a supplier/customer name?**
A: Yes! Click the pencil icon to edit.

**Q: What happens if I delete a supplier?**
A: The supplier is removed, but all historical purchases remain in the system.

**Q: Can employees see suppliers from other branches?**
A: No, employees only see their own branch. Owners see all branches.

**Q: Do I have to use the master list?**
A: No, you can still manually type names if needed. The auto-complete is optional.

**Q: Can I add the same supplier twice with different spellings?**
A: Yes, but it's not recommended. Stick to consistent naming for better reporting.
